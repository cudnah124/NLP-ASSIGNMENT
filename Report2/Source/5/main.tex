\subsection{Thành tựu chính}

\begin{itemize}
    \item \textbf{NER}: Fine-tune thành công BERT trên 6 nhãn thực thể pháp lý chuyên biệt với dataset 308 mẫu được annotation bằng pipeline Delta Operation.
    \item \textbf{SRL}: Tích hợp pretrained BERT-SRL từ HuggingFace Hub, ánh xạ PropBank arguments sang các vai nghĩa có ý nghĩa pháp lý.
    \item \textbf{Intent Classification}: Mô hình baseline TF-IDF + LogReg đạt accuracy 91\%; mô hình nâng cao DistilBERT cho khả năng hiểu ngữ cảnh tốt hơn.
\end{itemize}

\subsection{Hạn chế và hướng phát triển}

\begin{itemize}
    \item \textbf{Hạn chế}: Dataset NER còn nhỏ (308 mẫu); mô hình SRL pretrained chưa được fine-tune trên domain pháp lý; Intent classification chưa xử lý được các mệnh đề có nhiều intent cùng lúc.
    \item \textbf{Hướng phát triển}:
    \begin{itemize}
        \item Mở rộng dataset NER lên 1000+ mẫu bằng pipeline LLM synthetic.
        \item Fine-tune SRL trên legal corpus (CUAD, ContractNLI).
        \item Tích hợp output của BTL2 vào hệ thống hỏi đáp tự động (BTL3 -- RAG Chatbot).
    \end{itemize}
\end{itemize}
